# Instalacion 
- Node.js 6.5+
- MySQL 8 +
- npm 4 + 
- Angular cli 
- Angular 8.x
- Ionic 2.x
# Ejecutar un proyecto 
Una vez instalados todos los programas necesarios,solo necesitas clonar el repositorio:

***gh repo clone <name>***
  
Despues de ello entra a nuestro proyecto y ejecutalo:
 *npm start*
